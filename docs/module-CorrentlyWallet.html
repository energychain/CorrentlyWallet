(inner) newDemand(options)

Request new Demand Link - This method might be used to tell an energy provider to publish a new deman link for this account. Typical usage is to set email and provider in options. The given energy provider will get in contact with you to negotiate an offer. As soon as an energy contract is in place a demanLink will be published and could be used with the function linkDemand.

Typical Options:

  • email: communication address for contract, offer negotiation. Only allowed to be used for this Communication
  • provider: Might be 'stromdao' as contact persion (request will be routed to this provider)
  • yearlyDemand: Kilo-Watt-Hours per Year requested
  • address: Geo-Coded Address for point of consumption
Parameters:
Name Type Description
options object

Options required for energy provider to create a demandLink
